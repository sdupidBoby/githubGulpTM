# githubGulpTM
基于gulp构建mobileWeb项目

# 使用
1.下载githubGulpTM

2.cd到githubGulpTM目录，执行 npm install 安装package.json中已经配制好插件（如下

    "browser-sync": "^2.23.6",
    "del": "^3.0.0",
    "gulp": "^3.9.1",
    "gulp-autoprefixer": "^4.1.0",
    "gulp-clean": "^0.4.0",
    "gulp-clean-css": "^3.9.2",
    "gulp-concat": "^2.6.1",
    "gulp-htmlmin": "^4.0.0",
    "gulp-imagemin": "^4.1.0",
    "gulp-rev": "^8.1.1",
    "gulp-rev-collector": "^1.2.4",
    "gulp-stylus": "^2.7.0",
    "gulp-uglify": "^3.0.0",
    "imagemin-pngquant": "^5.0.1",
    "run-sequence": "^2.2.1",
    "vinyl-paths": "^2.1.0"
    
3.终端执行 gulp watcher ,启动项目

4.在dist目录下开发
